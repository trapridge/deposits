# tesseract

Tested on Node 14.9.0 and built with TypeScript and Fastify.

## Available Scripts

In the project directory, you can run:

### `npm run dev`

To start the app in dev mode. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm start`

For production mode

### `npm run test`

Run the test cases.

### `npm run prettier`

Reformat code.
